## 🌍 About This Project

This is **Project #01 inside The Winkieverse** a complete **Memory Card Matching Game** built using:

- HTML  
- CSS  
- Pure JavaScript (Vanilla JS)  

No frameworks.  
No libraries.  
Just logic, structure, and creativity.

This project focuses on mastering **core JavaScript fundamentals** through real implementation.

## 🎮 Game Overview

The Memory Card Game allows users to:

- 🎴 Flip two cards at a time  
- 🧠 Match identical emoji pairs  
- 📊 Track total moves  
- ⏳ Track time taken  
- 🏆 Win when all cards are matched  

The board is dynamically generated based on dimension size.

## ⚙️ How the Logic Works (High Level)

### 1️⃣ Board Generation
- Random emojis are selected.
- Cards are duplicated to create pairs.
- Cards are shuffled using Fisher-Yates algorithm.
- Game board is dynamically injected into the DOM.

### 2️⃣ Game State
The game tracks:
- `gameStarted`
- `flippedCards`
- `totalFlips`
- `totalTime`

### 3️⃣ Card Flipping System
- On click → card flips.
- If two cards are flipped:
  - If matched → mark as matched.
  - If not matched → flip back after 1 second.

### 4️⃣ Win Detection
When no unflipped cards remain:
- Timer stops
- Win screen appears
- Final moves and time are displayed
